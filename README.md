# [Chirag Yadav](https://chiragthukran.vercel.app/)

Here lies the source code for my personal portfolio.

Built with Next.js, React, Tailwind CSS, and Framer Motion.

## Features
- **Profile Details**: Information about my experience, education, and skills.
- **Project Catalog**: Custom components detailing my major Full Stack projects (Recipe Sharing, CPU Simulator, Velvo).
- **Dynamic Integrations**: Real-time GitHub and LeetCode activity tracking directly embedded into the UI.

## Local Development
To run this project locally:

```bash
pnpm install
pnpm run dev
```

Open [http://localhost:1408](http://localhost:1408) with your browser to see the result.
